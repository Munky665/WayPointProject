Click on the picture box to add nodes
add weight by selecting two nodes and inputing a number in the weight text box
From the file menu click Save to save the project you are working on (note: this file cannot be imported by other programs)
To open select open from the file menu
To export click the Export button on the bottom right of the window, this will export as .CSV
To Import CSV file click import from the bottom right of the window.
To add new map drag and drop .PNG file into picture box. this will clear all waypoints, please insure you have saved.
